## ⚙️ How to assemble a Reel (you'll do this for each one)

You have 3 options:

### Option A — DIY in CapCut (free, ~10 min/Reel)
1. Install **CapCut** (iOS / Android / Web)
2. Tap **New project** → import the generated cover image (`.png` from `instagram-content/reels/images/`)
3. Hold it on screen for 1 second
4. Add 3–4 short clips (10–15s total). Film with your phone:
   - The sticker being peeled
   - The sticker placed on laptop / water bottle / fridge
   - Pan over a person walking past wearing the tee
   - Anything Indian-context: chai, autorickshaw, monsoon street
5. Drop end with the cover image again + hashtags overlay
6. Add music — see "Music picks" below for each Reel
7. Export 1080×1920 (9:16) → save → Upload to Instagram

### Option B — Canva template (5 min)
1. Open canva.com → New design → **Instagram Reel (9:16)**
2. Drag the cover image as background
3. Add trending audio from Canva's library
4. Add 2–3 text overlays from the script below
5. Export → upload

### Option C — Hire a Fiverr editor (₹500–1500 per Reel, ~24 hr turnaround)
- Search Fiverr: "Instagram reel editor India"
- Send them: cover image + the script below + your product photos
- They return a ready-to-upload .mp4

## 🎯 Reel virality tips (read this once)

1. **First 1.5 seconds is everything.** The cover image you start with must SHOCK or INTRIGUE. Don't waste them on intros.
2. **Caption first line matters** — Instagram cuts after 30 chars on feed. Make it punchy.
3. **Add 8–12 hashtags** mixing: niche (#CJPDrip), mid (#IndianStreetwear, 100K-500K posts), broad (#IndianMemes, 1M+ posts).
4. **Reply to every comment in first hour** — IG boosts Reels with high engagement velocity.
5. **Share to Story 5 min after posting** — boosts initial reach.
6. **Use captions** (the auto-transcribe Instagram feature) — 80% of viewers watch with sound off.
7. **Loop the Reel** — end frame = first frame so it auto-loops. Higher watch time = higher reach.
8. **DM 5 friends to comment within first 5 min** — early engagement triggers the algorithm.
